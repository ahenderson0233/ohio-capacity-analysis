Utility

Network

Load uses each utility's load hosting-capacity layer; storage uses the published battery/BESS layer where one exists. FirstEnergy publishes a native battery (BESS) layer; AEP Ohio, AES Ohio and Duke Energy Ohio have no published BESS layer, so their generation hosting-capacity layer is used as a storage proxy and every popup says so. Values are available capacity in MW. In Combined, each active utility's layer is shown colored by available MW.

Capacity legend (MW)

Service territory

Each active utility's served area is tinted in its own color (clipped to Ohio) so utilities are visually distinguishable; the navy Ohio border is the hard analysis boundary — capacity, substations, and candidate sites are clipped to it, and nothing outside Ohio is shown. The checkbox toggles the colored footprints/border; the data clip stays applied regardless.

Substations

Substations come from HIFLD + OSM, ≥115 kV always shown, clipped to Ohio. The two sources are merged and de-duplicated (OSM within 200 m of a HIFLD/OSM substation is dropped) so each physical yard is counted once. Transmission-scale (≥115 kV) are always shown; lower-voltage points are filtered to the active service territory. Utility substations are merged into the same layer (same marker); their popup adds the published available-capacity breakdown.

Transmission lines

Every In-Service transmission line within Ohio — all utilities — drawn with real line geometry and colored by kV class. Click a line for details.

Candidate sites

Schools are the demand-side site universe: large, distributed, already-served parcels. Only schools inside the active service territory (where capacity data exists) are shown/loaded.

Dual-use eligibility

A site qualifies for large load + battery only when a Load segment ≥ load-min and a Storage segment ≥ storage-min lie within the eligibility distance on the same feeder (one point of interconnection). The matched feeder ID is shown in each site's breakdown.

Site eligibility (optional)

Screens the upstream bottleneck: a site qualifies only if its nearest substation publishes available load hosting capacity ≥ this value (AEP/AES/FirstEnergy; Duke omitted).
Substation & transmission criteria are optional — leave blank to not require them. Nearest distances are always computed & exported.

Applied after the capacity match, so it stacks with the feeder & substation filters. Ohio has no statewide parcel class, so this is usually inactive.
